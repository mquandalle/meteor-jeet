# Jeet

A grid system using CSS pre-processors

## Installation

meteor-jeet is installable from atmosphere:

```sh
$ meteor add mquandalle:jeet
```

## Usage

This package uses a new API to handle CSS pre-processors that was introduced in
Meteor v1.2. It won't work if you use a previous version of Meteor.

Once you installed this package, you can include the jeet jeet mixins in your
stylus stylesheet by using the following import statement:

```stylus
@import '{mquandalle:jeet}/jeet'
```
